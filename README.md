# Home Price Prediction: EDA and Regression Modeling

## 📌 Project Overview
This project explores a housing dataset to understand the factors that influence home prices and builds multiple regression models to predict sale prices. The workflow follows a full data science lifecycle, including exploratory data analysis (EDA), preprocessing, feature engineering, model training, and evaluation.

The goal is to compare different regression approaches and evaluate their performance in predicting home prices.

---

## 🧠 Objectives
- Perform exploratory data analysis to understand distributions and relationships
- Clean and preprocess the dataset
- Engineer and scale features for modeling
- Train and compare multiple regression models
- Evaluate model performance using standard regression metrics
